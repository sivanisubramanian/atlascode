ATlas — Athletic Training Learning System
ATlas is a structured learning platform built for student athletic trainers. It provides a clear curriculum, interactive lessons, and quick reference tools for students who are learning independently alongside a supervising AT.

What It Does
Student athletic trainers often have no structured curriculum to follow. They're expected to learn on the job, but supervising ATs are too busy to hand-hold, and there's no single resource that tells students what to learn or in what order.
ATlas solves this by providing:

Structured lesson modules organized by level (Level 1: Foundational, Level 2: Basic Skills, Level 3: Advanced)
Interactive lessons with definitions, diagrams, and embedded videos
Match games for reviewing terms and definitions
A Quick Reference page for fast sideline lookups (concussion red flags, heat illness, when to call 911, wound care, taping, common injury signs)
Progress tracking so students know what they've completed

Authentication & Class Management
ATlas uses Supabase for authentication and data storage.
Students sign up with a class code provided by their AT, access all lesson modules, and have their progress saved automatically between sessions.
Athletic Trainers sign up to receive a unique class code, share it with their students, and view a dashboard showing each student's progress across all modules.

About
Built by a high school student athletic trainer to fill a gap that every student AT experiences — no roadmap, no curriculum, no structured way to learn independently. Designed to be used alongside a supervising certified AT.
